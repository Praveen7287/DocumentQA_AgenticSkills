# DocumentQA_AgenticSkills
Document QA system is built around three modular skills:  document_preprocessor: Specializes in extracting and chunking text from various formats  query_retriever: Masters semantic search and relevance ranking  focused_summarizer: Excels at generating query-specific answers

A lightweight application for querying and summarizing documents using AI agent skills.

## Features

- **Multi-format support**: PDF, DOCX, TXT, MD
- **Semantic search**: Find relevant content using embeddings
- **Intelligent chunking**: Smart document splitting with overlap
- **Ollama integration**: Local LLM for privacy and control
- **Hybrid search**: Combines semantic and keyword matching
